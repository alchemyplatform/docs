## Getting Blocks

Retrieves information from a particular block in the blockchain.

* [eth\_getBlockByHash](/reference/eth-getblockbyhash)
* [eth\_getBlockByNumber](/reference/eth-getblockbynumber)
* [eth\_blockNumber](/reference/eth-blocknumber)
* [eth\_getBlockTransactionCountByHash](/reference/eth-getblocktransactioncountbyhash)

## Reading Transactions

Retrieves information on transactions within a block or the state of the blockchain.

* [eth\_getTransactionByHash](/reference/eth-gettransactionbyhash)
* [eth\_getTransactionByBlockHashAndIndex](/reference/eth-gettransactionbyblockhashandindex)
* [eth\_getTransactionByBlockNumberAndIndex](/reference/eth-gettransactionbyblocknumberandindex)
* [eth\_getTransactionReceipt](/reference/eth-gettransactionreceipt)
* [eth\_getTransactionCount](/reference/eth-gettransactioncount)

## Writing Transactions & EVM Execution

Allows developers to write data on-chain, and interact with smart contracts.

* [eth\_sendRawTransaction](/reference/eth-sendrawtransaction)
* [eth\_call](/reference/eth-call)

## Account Information

Returns information regarding an address's stored on-chain data.

* [eth\_getCode](/reference/eth-getcode)
* [eth\_getBalance](/reference/eth-getbalance)

## Event Logs

Returns logs, which are records that denote specific events within a smart contract.

* [eth\_getLogs](/reference/eth-getlogs)

## Chain Information

Returns information on the network and internal settings.

* [eth\_chainId](/reference/eth-chainid)
* [net\_version](/reference/net-version)

## Gas Estimation

Returns information on network gas estimates.

* [eth\_estimateGas](/reference/eth-estimategas)
* [eth\_gasPrice](/reference/eth-gasprice)

## Web3

Returns network configuration information.

* [web3\_clientVersion](/reference/web3-clientversion)
* [web3\_sha3](/reference/web3-sha3)

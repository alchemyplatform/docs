## Getting Blocks

Retrieves information from a particular block in the blockchain.

* [eth\_getBlockByHash](/reference/eth-getblockbyhash-bnb)
* [eth\_blocknumber](/reference/eth-blocknumber-bnb)
* [eth\_getBlockByNumber](/reference/eth-getblockbynumber-bnb)
* [eth\_getBlockTransactionCountByHash](/reference/eth-getblocktransactioncountbyhash-bnb)
* [eth\_getBlockTransactionCountByNumber](/reference/eth-getblocktransactioncountbynumber-bnb)

## Reading Transactions

Retrieves information on the state data for addresses regardless of whether it is a user or a smart contract.

* [eth\_getTransactionByHash](/reference/eth-gettransactionbyhash-bnb)
* [eth\_getTransactionByBlockHashAndIndex](/reference/eth-gettransactionbyblockhashandindex-bnb)
* [eth\_getTransactionByBlockNumberAndIndex](/reference/eth-gettransactionbyblocknumberandindex-bnb)
* [eth\_getTransactionReceipt](/reference/eth-gettransactionreceipt-bnb)
* [eth\_getTransactionCount](/reference/eth-gettransactioncount-bnb)

## Writing Transactions & EVM Execution

Allows developers to send ETH from one address to another, write data on-chain, and interact with smart contracts.

* [eth\_sendRawTransaction](/reference/eth-sendrawtransaction-bnb)
* [eth\_call](/reference/eth-call-bnb)

## Account Information

Returns information regarding an address's stored on-chain data.

* [eth\_getCode](/reference/eth-getcode-bnb)
* [eth\_getBalance](/reference/eth-getbalance-bnb)
* [eth\_getStorageAt](/reference/eth-getstorageat-bnb)

## Event Logs

Returns logs which are records that denote/provide context on specific events within a smart contract, like a token transfer or a change of ownership, for example.

* [eth\_getLogs](/reference/eth-getlogs-bnb)

## Chain Information

* [eth\_chainId](/reference/eth-chainid-bnb)
* [net\_listening](/reference/net-listening-bnb)
* [net\_version](/reference/net-version-bnb)

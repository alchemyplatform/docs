## Getting Blocks

Retrieves information from a particular block in the blockchain.

* [eth\_getBlockTransactionCountByHash](/reference/eth-getblocktransactioncountbyhash-astar)
* [eth\_getBlockTransactionCountByNumber](/reference/eth-getblocktransactioncountbynumber-astar)
* [eth\_blocknumber](/reference/eth-blocknumber-astar)
* [eth\_getBlockByNumber](/reference/eth-getblockbynumber-astar)
* [eth\_getBlockByHash](/reference/eth-getblockbyhash-astar)

## Reading Transactions

Retrieves information on the state data for addresses regardless of whether it is a user or a smart contract.

* [eth\_getTransactionByHash](/reference/eth-gettransactionbyhash-astar)
* [eth\_getTransactionReceipt](/reference/eth-gettransactionreceipt-astar)
* [eth\_getTransactionCount](/reference/eth-gettransactioncount-astar)
* [eth\_getTransactionByBlockHashAndIndex](/reference/eth-gettransactionbyblockhashandindex-astar)
* [eth\_getTransactionByBlockNumberAndIndex](/reference/eth-gettransactionbyblocknumberandindex-astar)

## Writing Transactions & EVM Execution

Allows developers to both send ETH from one address to another, write data on-chain, and interact with smart contracts.

* [eth\_call](/reference/eth-call-astar)
* [eth\_sendRawTransaction](/reference/eth-sendrawtransaction-astar)

## Account Information

Returns information regarding an address's stored on-chain data.

* [eth\_accounts](/reference/eth-accounts-astar)
* [eth\_getStorageAt](/reference/eth-getstorageat-astar)
* [eth\_getCode](/reference/eth-getcode-astar)
* [eth\_getBalance](/reference/eth-getbalance-astar)

## Event Logs

Returns logs which are records that denote/provide context on specific events within a smart contract, like a token transfer or a change of ownership, for example.

* [eth\_getLogs](/reference/eth-getlogs-astar)
* [eth\_newFilter](/reference/eth-newfilter-astar)
* [eth\_newPendingTransactionFilter](/reference/eth-newpendingtransactionfilter-astar)
* [eth\_newBlockFilter](/reference/eth-newblockfilter-astar)
* [eth\_getFilterChanges](/reference/eth-getfilterchanges-astar)
* [eth\_getFilterLogs](/reference/eth-getfilterlogs-astar)
* [eth\_uninstallFilter](/reference/eth-uninstallfilter-astar)

## Chain Information

Returns information on the Ethereum network and internal settings.

* [net\_version](/reference/net-version-astar)
* [eth\_chainId](/reference/eth-chainid-astar)

## Getting Uncles

Returns information on uncle blocks which are network rejected blocks and replaced by a canonical block instead.

* [eth\_getUncleByBlockNumberAndIndex](/reference/eth-getunclebyblocknumberandindex-astar)

## Gas Estimation

* [eth\_estimateGas](/reference/eth-estimategas-astar)
* [eth\_maxPriorityFeePerGas](/reference/eth-maxpriorityfeepergas-astar)
* [eth\_gasPrice](/reference/eth-gasprice-astar)

## Web3

Returns Ethereum network configuration information.

* [web3\_clientVersion](/reference/web3-clientversion-astar)
* [web3\_sha3](/reference/web3-sha3-astar)

## Real-time Events

Introduces WebSocket-based requests/responses which leverage a network connection allowing developers to listen for changes continuously without the need for HTTP polling.

* [eth\_subscribe](/reference/eth-subscribe-astar)
* [eth\_unsubscribe](/reference/eth-unsubscribe-astar)

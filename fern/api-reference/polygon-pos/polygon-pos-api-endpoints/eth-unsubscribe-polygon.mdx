# eth\_unsubscribe - Polygon PoS

Unsubscribe from different Polygon event types with a regular RPC call with eth\_unsubscribe as the method and the `subscriptionId` as the first parameter.

### Parameters

1. Subscription ID, as previously returned from an [`eth_subscribe`](/reference/eth-subscribe-polygon) call.

### Returns

`true` if a subscription was successfully canceled, or `false` if no subscription existed with the given ID.

### Example

#### Request

<CodeGroup>
  ```shell wscat
  wscat -c wss://polygon-mainnet.g.alchemy.com/v2/your-api-key

  {"jsonrpc":"2.0", "id": 1, "method": "eth_unsubscribe", "params": ["0x9cef478923ff08bf67fde6c64013158d"]}
  ```

  ```javascript alchemy-sdk
  // npm install alchemy-sdk
  import { Alchemy, Network } from "alchemy-sdk";

  const settings = {
    apiKey: "demo", // Replace with your Alchemy API Key.
    network: Network.MATIC_MAINNET, // Replace with your network.
  };
  const alchemy = new Alchemy(settings);

  alchemy.ws.removeAllListeners();
  ```
</CodeGroup>

<CodeGroup>
  ```json json
  {
      "jsonrpc":"2.0",
      "id":1,
      "result":true
  }
  ```
</CodeGroup>

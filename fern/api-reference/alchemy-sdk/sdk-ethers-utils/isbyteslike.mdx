Learn about the `isBytesLike` utils method.

# Introduction

The [`isBytesLike`](https://docs.ethers.org/v5/api/utils/bytes/#utils-isBytesLike) method is used to check if a value is a "bytes-like" object.

# Usage

This method should be used when you want to check if a value is a "bytes-like" object. This means that the value must be a `string`, an `array`, or a `Uint8Array`.

Here's a simple code example to demonstrate how to use this method:

<CodeGroup>
  ```javascript Javascript
  const { Utils } = require("alchemy-sdk");

  let stringValue = "0x010203";

  let isBytesLike = Utils.isBytesLike(stringValue); 
  console.log(isBytesLike); // true
  ```
</CodeGroup>

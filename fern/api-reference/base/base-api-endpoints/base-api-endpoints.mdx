## Getting Blocks

Retrieves information from a particular block in the blockchain.

* [eth\_blockNumber](/reference/eth-blocknumber-base)
* [eth\_getBlockByNumber](/reference/eth-getblockbynumber-base)
* [eth\_getBlockByHash](/reference/eth-getblockbyhash-base)
* [eth\_getBlockTransactionCountByNumber](/reference/eth-getblocktransactioncountbynumber-base)
* [eth\_getBlockTransactionCountByHash](/reference/getblocktransactioncountbyhash-base)

## Reading Transactions

Retrieves information on the state data for addresses regardless of whether it is a user or a smart contract.

* [eth\_getTransactionByBlockHashAndIndex](/reference/eth-gettransactionbyblockhashandindex-base)
* [eth\_getTransactionByBlockNumberAndIndex](/reference/eth-gettransactionbyblocknumberandindex-base)
* [eth\_getTransactionCount](/reference/eth-gettransactioncount-base)
* [eth\_getTransactionByHash](/reference/eth-gettransactionbyhash-base)
* [eth\_getTransactionReceipt](/reference/eth-gettransactionreceipt-base)

## Writing Transactions & EVM Execution

Allows developers to both send ETH from one address to another, write data on-chain, and interact with smart contracts.

* [eth\_call](/reference/eth-call-base)
* [eth\_sendRawTransaction](/reference/eth-sendrawtransaction-base)

## Account Information

Returns information regarding an address's stored on-chain data.

* [eth\_getBalance](/reference/eth-getbalance-base)
* [eth\_accounts](/reference/eth-accounts-base)
* [eth\_getProof](/reference/eth-getproof-base)
* [eth\_getCode](/reference/eth-getcode-base)
* [eth\_getStorageAt](/reference/eth-getstorageat-base)

## Event Logs

Returns logs which are records that denote/provide context on specific events within a smart contract, like a token transfer or a change of ownership for example.

* [eth\_getLogs](/reference/eth-getlogs-base)
* [eth\_newFilter](/reference/eth-newfilter-base)
* [eth\_newBlockFilter](/reference/eth-newblockfilter-base)
* [eth\_newPendingTransactionFilter](/reference/eth-newpendingtransactionfilter-base)
* [eth\_getFilterLogs](/reference/eth-getfilterlogs-base)
* [eth\_getFilterChanges](/reference/eth-getfilterchanges-base)
* [eth\_uninstallFilter](/reference/eth-uninstallfilter-base)

## Chain Information

Returns information on the Base network and internal settings.

* [eth\_protocolVersion](/reference/eth-protocolversion-base)
* [eth\_chainId](/reference/eth-chainid-base)
* [net\_listening](/reference/net-listening-base-1)
* [eth\_syncing](/reference/eth-syncing-base)
* [net\_version](/reference/net-version-base-1)

## Getting Uncles

Returns information on uncle blocks which are network rejected blocks and replaced by a canonical block instead.

* [eth\_getUncleByBlockHashAndIndex](/reference/eth-getunclebyblockhashandindex-base)
* [eth\_getUncleByBlockNumberAndIndex](/reference/eth-getunclebyblocknumberandindex-base)
* [eth\_getUncleCountByBlockHash](/reference/eth-getunclecountbyblockhash-base)
* [eth\_getUncleCountByBlockNumber](/reference/eth-getunclecountbyblocknumber-base)

## Gas Estimation

* [eth\_estimateGas](/reference/eth-estimategas-base)
* [eth\_gasPrice](/reference/eth-gasprice-base)

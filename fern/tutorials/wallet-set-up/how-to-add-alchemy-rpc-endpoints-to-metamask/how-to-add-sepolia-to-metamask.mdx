# 3. Add Sepolia to Metamask

To start deploying smart contracts and interacting with test applications on the Sepolia test network, you need to connect your wallet to the Sepolia network.

To add the Sepolia testnet to Metamask, click the network button at the top of your wallet and click ‚ÄúAdd Network‚Äù.

![722](e0cf5c0-mm-sepolia.png "mm-sepolia.png")

At the bottom of the page, click ‚ÄúAdd a network manually‚Äù.

![2160](8a951a9-mm-adding-network.jpeg "mm-adding-network.jpeg")

Next, go to your Alchemy dashboard, click "View Key" and then copy the ‚ÄúHTTPS URL‚Äù.

![1418](c483dcc-sepolia-url.jpeg "sepolia-url.jpeg")

Go back to Metamask and paste your Sepolia RPC URL into the network configuration details along with the following information:

* **Network Name** - Sepolia Testnet
* **New RPC URL** - [https://eth-sepolia.g.alchemy.com/v2/\[YOUR-API-KEY](https://eth-sepolia.g.alchemy.com/v2/%5BYOUR-API-KEY)]
* **Chain ID** - 11155111
* **Currency Symbol** - SepoliaETH
